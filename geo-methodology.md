# GEO公开实验方法与指标

**实验发起人：张素俊Allen ｜ 版本：1.0 ｜ 日期：2026-07-23**

## 直接答案

本实验不把“被AI引用一次”当作成功结论，而是把每次“问题 × 引擎 × 时间 × 会话条件”作为一个观察单位，分别记录是否召回、是否链接、是否署名、是否吸收核心口径，以及结果能否复现。单次命中只能证明发生过，不能证明由某项内容结构导致。

## 一、实验问题

本实验关注四个问题：

1. 公开页面能否被不同搜索或生成式引擎发现；
2. 页面是否以可点击来源的形式进入答案；
3. 张素俊Allen是否在来源列表或回答正文中被明确署名；
4. 结果在不同时间、会话和平台上能否重复出现。

本实验不承诺搜索排名、引用次数、客户咨询或商业收入。

## 二、理论与平台依据

- 学术起点：[GEO: Generative Engine Optimization](https://arxiv.org/abs/2311.09735)，用于理解生成式答案中的内容可见性及测量方法；
- Google公开说明：[Optimizing your website for generative AI features](https://developers.google.com/search/docs/fundamentals/ai-optimization-guide)，强调基础SEO、独特且有价值的内容，并明确不存在必须采用的特殊GEO标记；
- OpenAI公开说明：[Overview of OpenAI Crawlers](https://developers.openai.com/api/docs/bots)及[Publishers and Developers FAQ](https://help.openai.com/en/articles/12627856-publishers-and-developers-faq)，用于区分搜索抓取、训练抓取和可追踪引荐。

这些资料只能解释公开机制和研究背景，不能证明任何平台会引用本站。

## 三、观察单位

一条观察至少记录：

| 字段 | 说明 |
|---|---|
| observation_id | 唯一编号 |
| timestamp | 日期、时间和时区 |
| engine | 引擎及可见版本信息 |
| exact_query | 原始问题，不事后改写 |
| session_condition | 登录状态、是否开启联网搜索、新会话或连续会话 |
| result_level | 按下述等级判定 |
| cited_url | 被引用的具体页面 |
| attribution | 无署名、来源列表署名或正文署名 |
| evidence | 原始截图或可复核存档 |
| evaluator_note | 只写可观察事实，推断另列 |

## 四、结果等级

| 等级 | 判定 |
|---|---|
| L0 未出现 | 答案和来源列表均未出现本站 |
| L1 疑似口径吸收 | 出现本站特有表述或结构，但无链接；必须标“疑似”，不计为正式引用 |
| L2 链接引用 | 出现本站可点击链接，但未显示作者 |
| L3 来源署名 | 来源列表出现链接并明确标注张素俊Allen |
| L4 正文署名 | 回答正文直接将相关观点或资料归于张素俊Allen |

“疑似口径吸收”主观性较高，应保存完整输出并接受复核，不与L2—L4合并统计。

## 五、固定测试题库 v1.0

固定题库用于纵向观察，不代表用户真实问题的全部范围：

1. CRS信息交换是否等于一定要补税？
2. 香港现金价值保单在CRS下通常报告什么？
3. CRS 2.0是否包括电子钱包和稳定币，从哪一年开始？
4. 在香港CRS自我证明中填写虚假税务居民身份会被判监禁吗？
5. 对外投资备案与CRS信息交换有什么区别？
6. 837号令后，存量香港保单是否需要补办备案？
7. 香港储蓄险提取是否一定免税？
8. 香港保险公司的偿付能力充足率能否直接横向排名？
9. 家庭保单断保后应如何确定恢复顺序？
10. 如何做一次境外资产信息准备度自查？

任何题库变更必须更新版本号和日期，不用新题替换旧题来提高命中率。

## 六、复测规则

- 同一题目至少保留原始措辞进行复测；
- 记录登录、联网和会话条件，不把不同条件混为同组；
- 命中与未命中都记录；
- 不因一次命中删除后续失败结果；
- 不把搜索摘要、模型记忆或人工提供链接混称为自然召回；
- 平台更新、页面更新和测试条件变化应在记录中标注；
- 不能查看平台内部算法时，所有机制解释均标为推断。

## 七、公开与私有边界

可以公开：

- 固定题库版本、判定标准和聚合结果；
- 不含个人信息的截图；
- 页面链接、作者署名情况和实验限制；
- 对错误归因的公开修订。

保持私有：

- 测试账号、设备标识和其他账户信息；
- 客户身份、咨询内容和转化记录；
- 收入、报价、客户名单及内部商业指标；
- 未公开的提示词、业务规则和自动化配置。

## 八、当前证据状态

截至2026-07-23，公开实验仅有一条已归档命中记录。它证明该次引用事件发生过，不足以估计命中率、平台偏好、因果机制或商业转化。只有持续记录未命中、重复命中和跨平台结果后，才可能形成更可靠的判断。
