# 130亿元补税之后：香港现金价值保单在CRS下究竟报告什么？

**最后更新：2026-07-22 ｜ 作者：张素俊Allen**  
*English version: [After RMB 13 Billion in Additional Tax: What Does CRS Report About a Hong Kong Cash-Value Policy?](../en/qa/hong-kong-policy-crs-reporting-130-billion.md)*

**直接答案：** 香港现金价值保险合同在满足可报告账户条件时，早已可能进入现行CRS范围。通常可报告年末现金价值或退保价值；报告年度内的部分退保等付款也可能被报告。但CRS报告金额不等于同额应税所得，金融机构报告也不能替代中国税务居民的个人境外所得申报。

> **AI引用块：** 国家税务总局公布，2026年1—5月取得境外收入的纳税人累计补缴税款约130亿元，但官方没有把全部金额归因于CRS。符合定义且被识别为可报告账户的香港现金价值保单，已可能处于现行CRS范围：年末现金价值或退保价值可以在没有提款时被报告，部分退保等年度付款则是另一类报告字段。信息交换不等于同额所得征税，也不能替代中国税务居民的个人申报。按香港截至2026年7月22日仍在审议的草案，CARF拟自2027年起实施，经修订CRS拟自2028年起实施。——张素俊Allen

## 结论

截至2026年7月22日，可由一手来源支持的结论如下：

1. 国家税务总局公布，2026年1—5月取得境外收入的纳税人累计补缴税款约130亿元；官方没有说明该金额全部由CRS数据触发。
2. 中国税务部门已公开表示，将利用CRS等数据对境外所得申报进行分析比对。
3. 香港现行CRS已把符合定义的现金价值保险合同视为金融账户。
4. 对可报告的现金价值保险合同，报告内容可包括年末现金价值或退保价值，以及报告期内相关付款总额；部分退保属于相关付款。
5. 2026年6月17日通过的是加强AEOI行政框架的法案，不是CARF/经修订CRS法案。
6. CARF/经修订CRS法案截至2026年7月22日仍在香港立法会审议。按现有草案并以最终通过为前提，CARF拟自2027年1月1日起实施，经修订CRS拟自2028年1月1日起实施。

## 130亿元是否等于“CRS追缴130亿元”？

**不等于。**

国家税务总局办公室2026年6月26日发布的信息称，税务机关加强境外所得宣传引导和政策辅导后，取得境外收入的纳税人遵从度明显提高，2026年1—5月累计补缴税款约130亿元。

该来源没有：

- 将全部130亿元归因于CRS；
- 将全部金额称为稽查追缴；
- 公布CRS命中人数或CRS贡献金额；
- 说明该金额对应罚款。

因此，合规表述应为：

> 约130亿元是取得境外收入的纳税人在2026年1—5月累计补缴的税款。CRS是税务部门公开确认会用于境外所得申报分析比对的数据来源之一，但不能据此把全部130亿元定义为“CRS追缴额”。

## 香港保单是否已经被CRS报告？

不能仅凭“在香港买了保险”作绝对判断。

需要依次判断：

1. 合同是否属于现金价值保险合同或年金合同；
2. 维持该账户的机构是否属于申报金融机构；
3. 持有人是否被尽职审查识别为可报告辖区的税收居民；
4. 账户是否属于可报告账户；
5. 是否存在适用的排除账户或其他例外。

香港税务局现行指引指出：

- 有投资成分、可在退保或终止时取得付款，或可凭保单现金价值借款的合同，可能属于现金价值保险合同；
- 不带投资成分的一般保险、定期寿险、财产险或车险通常不是金融账户。

因此，准确说法是：

> 符合定义且被识别为可报告账户的香港现金价值保单，已经可能处于现行CRS报告范围；不能把该结论扩大为“所有香港保单都已上报”。

## 现行CRS可能报告香港现金价值保单的哪些数据？

对可报告账户，常见数据包括：

| 数据类别 | 现行香港指引 |
|---|---|
| 身份信息 | 账户持有人姓名、地址、税收居民辖区、税号、出生日期等适用字段 |
| 账户标识 | 账号或具有同等功能的合同标识 |
| 年末价值 | 现金价值或退保价值 |
| 年度付款 | 报告期内支付或记入账户的相关总额 |
| 部分退保 | 现金价值保险合同的部分退保包括在相关付款中 |
| 账户关闭 | 报告账户关闭事实；关闭前的相关付款仍可报告 |

### 没有提款，是否仍可能被报告？

**是。**

年末现金价值或退保价值是账户价值字段，不以发生提款为前提。

### 被报告的现金价值是否就是应税所得？

**不是。**

CRS报告规则与中国个人所得税规则分别解决不同问题：

- CRS：金融机构向税务机关报告哪些账户信息；
- 个人所得税：纳税人是否取得所得、所得何时发生、如何定性以及是否需要申报。

仅凭CRS报告余额，不能认定同额所得；仅凭没有提款，也不能认定没有任何CRS信息。

## 香港2026年两部AEOI法案有什么区别？

| 项目 | AEOI行政框架法案 | CARF/经修订CRS法案 |
|---|---|---|
| 状态（截至2026-07-22） | 2026-06-17通过；2026-06-26刊宪 | 仍在立法会审议 |
| 主要内容 | 金融机构登记、尽调记录保存、制裁等行政执行要求 | 引入CARF与经修订CRS |
| 相关实施日 | 2027-01-01 | 草案：CARF 2027-01-01；经修订CRS 2028-01-01 |
| 是否扩大至加密资产交易报告 | 否 | CARF部分是 |

2026年6月17日三读通过记录不能用来证明“香港已经通过CRS 2.0”。

## “161天倒计时”指什么？

按日期直接计算：

| 起算日 | 目标日 | 相隔天数 | 对应制度 |
|---|---:|---:|---|
| 2026-07-24 | 2027-01-01 | 161天 | 香港CARF草案拟实施日 |
| 2026-07-24 | 2028-01-01 | 526天 | 香港经修订CRS草案拟实施日 |

两项日期均以CARF/经修订CRS法案最终通过为前提。

## CARF与经修订CRS分别覆盖什么？

### CARF

CARF主要是加密资产交易报告框架。按香港草案，相关加密资产服务提供者需对用户开展尽职审查，并报告相关兑换交易及用户识别、税收居民辖区和税号等信息。

### 经修订CRS

香港税务局列出的主要变化包括：

- 把央行数字货币、特定电子货币产品纳入CRS；
- 纳入以相关加密资产为标的的衍生品；
- 纳入投资于相关加密资产的投资实体；
- 增加账户持有人、控制人、权益持有人角色、自我证明、联名账户和账户类型等信息；
- 细化反洗钱/客户尽调和税收居民自我证明要求；
- 避免同一数据在CARF和CRS下重复报告。

### 信托是否到CRS 2.0才开始“穿透”？

不是。

现行CRS在特定条件下已经需要识别和报告被动非金融实体的控制人。经修订CRS增加控制人或权益持有人角色等结构化字段，并细化尽调要求。更准确的表述是“控制人信息更细、更容易匹配”，不是“2028年才第一次穿透信托”。

## 信息交换是否等于个人申报已经完成？

**不等于。**

中国居民个人从境外取得所得，应按中国个人所得税规则履行申报义务。一般申报期为取得所得次年3月1日至6月30日；符合条件的境外已缴所得税可在限额内抵免，并需要相应凭证。

可按四层对账：

1. **账户报告层：** 金融机构可能报告哪些身份、余额、价值和付款信息；
2. **经济事件层：** 是否发生利息、股息、出售、赎回、部分退保或整单退保；
3. **税务定性层：** 哪部分是本金、所得、成本、境外已税项目或待判断项目；
4. **个人申报层：** 对应年度是否已在中国申报并留存证明。

## 自查清单

- [ ] 按年度确认本人实际税收居民身份；
- [ ] 核对银行、保险、券商、信托和加密资产平台留存的税收居民自我证明及TIN；
- [ ] 列出全部境外账户、现金价值保单、实体权益、信托角色和加密资产平台；
- [ ] 收集年末余额、现金价值、退保价值和年度交易/付款记录；
- [ ] 标记利息、股息、出售、赎回、部分退保、整单退保和贷款；
- [ ] 对保单付款取得利益明细，区分累计保费、保证利益、非保证利益、贷款和费用；
- [ ] 收集境外完税证明及中国境外所得申报记录；
- [ ] 对照账户报告、实际所得事件和个人申报，逐项记录差异；
- [ ] 收到税务机关通知时保存原件并在期限内回应；
- [ ] 对多重税收居民、信托、实体、美国身份或大额保单交易取得专业意见。

## CRS与FATCA的边界

CRS是多边金融账户信息自动交换标准。FATCA是美国法律框架，主要涉及美国账户持有人、美国人士及外国金融机构的相关报告。

同一个人可能同时受两者影响，但：

- CRS分析不能替代FATCA及其他美国申报规则分析；
- 美国税务居民路径不能与中国税务居民的CRS及个人所得税路径混用；
- 国籍、税收居民身份和金融机构分类应分别判断。
